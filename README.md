# Data Analysis

Trabajos y proyectos del curso de Análisis de Datos — Universidad Mayor.
Excel, Power Query y dashboards en Power BI.

---

## Herramientas utilizadas

- **Microsoft Excel** — organización y estructuración de datos
- **Power Query** — transformación y modelado de datos
- **Power BI** — visualización y creación de dashboards

---

## Proyectos

### Trabajo Unidad 3 — Análisis de Ventas Contoso
Reporte de ventas desarrollado en Power BI usando el dataset ficticio de Microsoft (Contoso),
con datos de Latinoamérica y España.

**Archivos:**
- `Paz_Fernanda_Trabajo_U3.pbix` — reporte Power BI
- `Data_Source_Contoso_Spanish_America_-_Facturas.csv` — dataset fuente (18.569 registros)

**Contenido del análisis:**
- Ventas por producto, categoría y país
- Análisis por tipo de tienda y período de tiempo
- Métricas de precio, costo y cantidad

---

### Examen Final — Análisis de Dotación de Personal
Reporte de recursos humanos desarrollado en Power BI, con modelo estrella
construido mediante transformaciones en Power Query.

**Archivos:**
- `Paz_Fernanda_Examen.pbix` — reporte Power BI
- `BaseHiring_cut.xlsx` — dataset fuente (6 hojas)

**Modelo de datos:**
- `TablaEmpleados` (tabla de hechos) relacionada con 5 tablas de dimensión:
  Género, Etnia, Jornada, Remuneración y Grupo Etario

**Contenido del análisis:**
- Total de empleados por género, etnia y grupo etario
- Distribución por tipo de jornada y remuneración
- Evolución temporal de contrataciones
